Coin MarketCap semi-clone

I created this app during my fascination with cryptocurrencies, setting myself a challenge to build an intuitive platform. Utilizing SwiftUI and the MVVM architecture, I crafted the views, while the Coincap.io API provided real-time crypto prices. With the newly introduced Apple Swift Charts, I designed interactive charts to visualize market trends seamlessly. Moreover, users can maintain personalized portfolios, and CoreData ensures secure data persistence, enabling them to manage and monitor their investments effortlessly. This app offers a comprehensive and user-friendly solution for staying informed and making the most of crypto investments.

Tech Used:

SwiftUI Swift-Charts Core Data Combine CoinCap.io Api Data

Calculator App

I created a fully functional calculator app inspired by mechanical keyboards. Built with SwiftUI and following the MVVM architecture, it offers a history log with the ability to save calculations using Realm. Users can review and delete logs as needed. The app supports common arithmetic operations and dynamically adjusts to different iPhone sizes. The sleek design is complemented by dual mechanical keyboard-inspired themes, accessible through the dark mode toggle. Experience an intuitive and visually appealing calculator that combines form and function seamlessly.

Tech Used:

SwiftUI Realm database MVVM architecture

Chatty App

Chatty App is a secure messaging application with a simple and intuitive user interface. Users can authenticate themselves safely and engage in private conversations. While it may not have all the bells and whistles of a full-fledged chat app, Chatty App ensures a seamless and secure chatting experience. Built using UIKit and following the MVC architecture, the app integrates Firebase's Firestore for secure authentication and data storage.

Tech Used:

UIKit Firebase CocoaPods
